# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.5.1] - 2024-12-30

### Fixed
- **CircularProgress** - Fixed indeterminate animation by properly coordinating SVG rotation and stroke-dashoffset animations

### Changed
- Package version updated to 1.5.1

---

## [1.5.0] - 2024-12-30

### Added
- **Tabs** - Tabs component for organizing content into multiple panels with variants (default, pills, underline) and horizontal/vertical orientations
- **Toast** - Toast notification component for displaying temporary messages with multiple variants and customizable positions
- **Textarea** - Textarea component for multi-line text input with resize options, character count, and validation support
- **TimeInput** - Time input component with 12h and 24h format support, validation, and formatting
- **Tooltip** - Tooltip component that displays additional information on hover with multiple positions and delay support
- **User** - User component that displays user information with avatar, name, and description

### Documentation
- Updated README.md with comprehensive documentation for all 6 new components
- Added usage examples and code snippets for Tabs, Toast, Textarea, TimeInput, Tooltip, and User
- Updated component list and import examples

---

## [1.4.1] - 2024-12-30

### Fixed
- **ScrollShadow** - Fixed content overflow and layout issues, improved height constraint handling
- **Switch** - Fixed thumb (circle) vertical centering issue in the switch track

### Changed
- Improved ScrollShadow component structure for better containment and shadow overlay positioning
- Enhanced Switch component thumb positioning for proper vertical and horizontal alignment

---

## [1.4.0] - 2024-12-30

### Added
- **RadioGroup** - Radio button group component for single selection with flexible layout options
- **RangeCalendar** - Calendar component for selecting date ranges with visual range highlighting
- **ScrollShadow** - Component that adds scroll shadows to indicate scrollable content areas
- **Select** - Select/dropdown component with single or multiple selection support and custom styling
- **Skeleton** - Skeleton loading component for placeholder content with multiple variants and animations
- **Slider** - Slider component for selecting numeric values with min/max/step controls and marks
- **Snippet** - Code snippet component with copy functionality and customizable symbols
- **Spacer** - Spacer component for adding consistent spacing in vertical, horizontal, or both directions
- **Switch** - Toggle switch component with customizable size, color, and label support
- **Table** - Table component for displaying tabular data with sorting, striped rows, and hover effects

### Documentation
- Updated README.md with comprehensive documentation for all 10 new components
- Added usage examples and code snippets for RadioGroup, RangeCalendar, ScrollShadow, Select, Skeleton, Slider, Snippet, Spacer, Switch, and Table
- Updated component list and import examples

---

## [1.3.0] - 2024-12-30

### Added
- **Navbar** - Navigation bar component with brand and links, supports multiple variants (default, dark, light) and link positioning
- **NumberInput** - Number input component with increment/decrement buttons, min/max validation, and step control
- **Pagination** - Pagination component for navigating through pages with customizable visible page count and navigation controls
- **Popover** - Popover component that displays content on hover or click, supports multiple positions (top, bottom, left, right)
- **Progress** - Linear progress bar component with multiple variants (default, success, warning, error, info) and label support

### Documentation
- Updated README.md with documentation for all 5 new components
- Added usage examples and code snippets for Navbar, NumberInput, Pagination, Popover, and Progress
- Updated component list and import examples

---

## [1.2.0] - 2024-12-19

### Added
- **Calendar** - Calendar component for date selection with month and year navigation
- **Checkbox** - Checkbox input component with label support and indeterminate state
- **CheckboxGroup** - Group of checkboxes with controlled selection and flexible layout options
- **Chip** - Chip component for displaying labels, tags, or removable items with multiple variants
- **CircularProgress** - Circular progress indicator with determinate and indeterminate variants
- **Code** - Component for displaying code snippets inline or in blocks with syntax highlighting support
- **DateInput** - Text input component for date entry with customizable formatting
- **DatePicker** - Date picker component with calendar popup and date range restrictions
- **DateRangePicker** - Date range picker for selecting start and end dates
- **Divider** - Divider component for separating content horizontally or vertically with multiple styles
- **Dropdown** - Dropdown/select component with single or multiple selection support
- **Drawer** - Slide-out drawer component that can appear from any side (left, right, top, bottom)

### Documentation
- Updated README.md with comprehensive documentation for all 23 components
- Added usage examples and code snippets for all new components
- Updated component list and import examples

### Changed
- Package version updated to 1.2.0
